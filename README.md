# 📘 Introducción a NumPy

Bienvenidx a este repositorio donde exploramos NumPy, una librería esencial para la computación científica en Python. Aquí encontrarás ejemplos prácticos y explicaciones claras para comenzar a trabajar con arrays multidimensionales y operaciones matemáticas avanzadas.

---

## 🧪 Requisitos

Para ejecutar este proyecto, asegúrate de tener:

- Python 3.6 o superior
- [pip](https://pip.pypa.io/en/stable/) (gestor de paquetes de Python)

---

## 📦 Instalación

1. **Clona el repositorio:**

   ```bash
  git clone  https://github.com/CAMCDI/Simulacion
   cd Simulacion
````

2. **Crea un entorno virtual:**

   * En Windows:

     ```bash
     python -m venv venv
     .\venv\Scripts\activate
     ```

   * En macOS/Linux:

     ```bash
     python3 -m venv venv
     source venv/bin/activate
     ```

3. **Instala las dependencias:**

   ```bash
   pip install -r requirements.txt
   ```

4. **Inicia Jupyter Notebook:**

   ```bash
   jupyter notebook
   ```

   Esto abrirá Jupyter en tu navegador, donde podrás explorar y ejecutar los ejemplos.

---

## 🧠 Contenido del Proyecto

* **Arrays en NumPy:** Aprende sobre arrays unidimensionales y multidimensionales.
* **Creación de Arrays:** Genera arrays con valores específicos, ceros, unos o aleatorios.
* **Acceso y Modificación:** Accede y modifica elementos de arrays utilizando índices y slicing.
* **Operaciones Aritméticas:** Realiza operaciones matemáticas como suma, resta, multiplicación, división y más.
* **Visualización:** Genera gráficos con Matplotlib para visualizar datos.
